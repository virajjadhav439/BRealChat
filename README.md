# BRealChat

A real-time chat application built with the MERN Stack and Socket.IO.
This is My Second Internship Project.

## Features

- JWT Authentication
- Real-Time Messaging
- Online/Offline Status
- Typing Indicators
- Persistent Chat History
- Responsive UI
- Dark/Light Theme

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT
