# BRealChat

A full-stack real-time messaging application built using the MERN Stack and Socket.IO.

## 🚀 Features

- JWT Authentication
- Google OAuth Authentication
- Real-Time Messaging
- Online / Offline Presence
- Typing Indicators
- Message Read Receipts
- Persistent Chat History
- Responsive Mobile-Friendly UI
- Dark / Light Theme
- Modern Landing Page

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Socket.IO Client
- Framer Motion
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT
- Google OAuth
