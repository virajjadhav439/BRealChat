# BRealChat

A full-stack real-time messaging application built using the MERN Stack and Socket.IO.

## 🚀 Features

- JWT Authentication
- Google OAuth
- Real-Time Messaging
- End-to-End Encryption (OpenPGP)
- Online/Offline Status
- Typing Indicators
- Message Seen Indicators
- Persistent Chat History
- Responsive UI
- Dark/Light Theme

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Socket.IO Client
- Framer Motion
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT
- Google OAuth
